Inicio

Desarrollado en Flask junto con un servidor HAPI FHIR

Flask es un framework web ligero y flexible para Python que se utiliza para desarrollar aplicaciones web. Proporciona herramientas y bibliotecas para crear rápidamente aplicaciones web y APIs.

HAPI FHIR es una implementación en Java del estándar FHIR (Fast Healthcare Interoperability Resources). FHIR es un estándar de datos para el intercambio y la interoperabilidad de la información de salud en el campo de la atención médica.
